# Proyecto implementacion PokeAPI

Este proyecto es una aplicación desarrollada en Laravel versión v13.20.0 para la busqueda de Pokemon mediante la API PokeAPI

## Dependencias

El proyecto utiliza varios paquetes de Composer, incluyendo:

- mockery/mockery:1.6.12
- psr/log:3.0.2
- fakerphp/faker:v1.24.1
- myclabs/deep-copy:1.13.4
- symfony/routing:v8.1.0
- phpstan/phpdoc-parser:2.3.3
- laravel/tinker:v3.0.2
- symfony/console:v8.1.1
- symfony/mailer:v8.1.1
- laravel/framework:v13.20.0
- guzzlehttp/guzzle:7.14.2

Entre otras.

El proyecto también utiliza varios paquetes de JavaScript, como:

- @tailwindcss/vite: 4.0.0,
- tailwindcss: 4.0.0

Estos paquetes son administrados por el gestor de paquetes npm.

## Instalación

Para instalar el proyecto, sigue estos pasos:

1. Clona el repositorio
2. Navega a la carpeta del proyecto
3. Ejecuta el comando `composer install` para instalar las dependencias de PHP

## Uso

Una vez que haya instalado el proyecto, puede ejecutarlo usando el comando adecuado de su servidor local. la ruta principal es la base (/) y una ruta POST para mostrar los resultados de las busquedas
